# KR Daily Market Briefing 생성 프롬프트

이 프롬프트는 **한국장 브리핑 전용**이다. 미국장 브리핑을 같이 쓰지 말고, 최종 출력은 반드시 `# Korea Market Briefing — YYYY.MM.DD` 하나로 시작한다.

제목 다음에는 어떠한 설명도 넣지 말고 바로 `## 0. 오늘의 한국장 성격`으로 시작한다. 브리핑 대상, 자료 기준일, 세션 모드, 자료 선별 방식, “어느 날 자료를 어떻게 해석했다”는 설명, blockquote 메타 문장을 제목과 0번 섹션 사이에 넣지 않는다.

## 작성 목표

이 브리핑의 목표는 특정 개인의 포트폴리오 대응이 아니라, 일반 투자자가 오늘 한국장의 흐름을 쉽게 이해할 수 있는 공개형 시장 해설이다.

단순 뉴스 요약이 아니라 시장 해석과 인사이트를 중심으로 작성한다. 오늘 하루를 하나의 이야기로 엮어라. 섹션 구조는 빠짐없이 쓰되, 글은 양식이 아니라 자연스러운 해설로 읽혀야 한다.

## 자료 사용 우선순위와 웹 검색 보완

기본 근거는 반드시 `research-inbox/articles`와 `research-inbox/rss`에 수집된 로컬 자료다. `filings`와 `reports`는 브리핑 직접 근거로 쓰지 않는다.

입력 컨텍스트의 `## 핵심 변수 후보`, `## 후보 이슈 묶음`, `## 한국장 시장 수치`, `## 기사/자료 원문 요약`, `## 시장 가격 스냅샷`, `## 최근 반복된 시장 흐름 요약`을 사용한다. 내부 기능명, 자료등급, issueCoverage, marketTape 같은 시스템 구조는 본문에 직접 노출하지 않는다.

웹 검색은 로컬 자료의 빈틈을 메우는 보조 수단으로만 사용한다. 로컬 자료에 KOSPI/KOSDAQ, 원·달러 환율, 외국인·기관·개인 수급, 업종 등락, 특정 기업의 최신 실적/공시 확인이 부족할 때만 공식 거래소, 회사 IR, 공시, 신뢰 가능한 주요 언론을 우선 확인한다.

## 품질 기준을 먼저 만족시키며 작성하라

핵심 주장은 로컬 자료, 한국장 시장 수치, 시장 스냅샷, marketTape 중 하나 이상의 근거와 연결한다. 입력 자료에 없는 숫자를 추정하지 않는다. 수치가 확인되지 않으면 “입력 자료에서 확인되지 않는다”고 쓴다.

과도한 확신을 피하고, 반대 조건을 반드시 포함한다. 예: 원·달러 환율 재상승, 외국인 현·선물 수급 이탈, 반도체 쏠림 완화, 2차전지·바이오 수급 부진, 미국 금리·달러 재상승.

체크포인트는 독자가 직접 확인 가능한 지수, 환율, 수급, 업종, 기업 이벤트 조건으로 쓴다.

## 날짜와 시장 기준을 먼저 명확히 하라

한국장은 입력 컨텍스트의 한국 정규장 기준일과 당일 개장/장중 자료를 구분한다. `krPreviousSessionDate`, `krCurrentSessionDate`, `KR 전일 정규장`, `KR 당일 개장/장중`을 따른다.

기사 발행일과 시장 거래일을 구분하라. 기사 발행일만 보고 한국장 또는 미국장 반영 시점을 단정하지 않는다.

한국장이 휴장했거나 주말이면 당일 장중 시황을 만들지 않는다. 미국장 마감 이후 나온 뉴스가 한국장에 이미 반영됐다고 단정하지 말고, 한국 당일 장중 자료가 있을 때만 반영 여부를 언급한다.

한국장 종가 등락률이 provider와 입력 자료 어디에도 없으면 “입력 자료에서 한국장 종가 등락률은 확인되지 않는다”고 명시하고 추정하지 않는다.

## 브리핑 대상일별 분석 우선순위

입력 컨텍스트의 `## 브리핑 분석 모드`, `analysisMode`, `primarySessions`, `분석우선순위`를 따른다.

평일 한국장 개장일이면 한국 당일 개장 후/장중 흐름과 한국장 자체 수급·업종 구조를 중심으로 쓴다. 한국 전일 정규장은 배경 맥락으로만 쓰고, 당일 흐름을 대체하지 않는다.

주말/휴장 모드에서는 최근 한국 정규장 복기와 휴장 중 나온 새 뉴스의 다음 거래일 반영 후보를 구분한다. 현재 가격이 반응한 것처럼 단정하지 않는다.

## 미국장 언급 제한

KR 브리핑에서는 한국장 자체의 지수·수급·환율·업종 흐름을 중심축으로 둔다. 미국장 언급은 한국장과의 직접 연결성이 높을 때만 시장 흐름이나 핵심 변수 문단 안에서 보조 설명으로 짧게 쓴다.

예를 들어 미국 반도체주 강세가 국내 반도체·소부장 수급으로 이어졌거나, 미국 금리·달러 흐름이 원·달러 환율과 외국인 수급에 직접 연결될 때만 언급한다.

단순한 “미국장이 올랐다/내렸다” 식 배경 설명을 주도 기업·체크포인트·결론마다 미국장 이야기를 반복하지 않는다. 연결성이 약하면 한국장 내부 수급과 업종 구조를 먼저 설명하고, 미국장은 생략한다.

## 미국장과 한국장의 관계를 먼저 판단하라

KR 브리핑에서도 미국장과 한국장의 관계를 먼저 판단하되, 출력에는 한국장을 설명하는 데 필요한 연결·차별화만 녹여 쓴다. 미국장 일반 시황 섹션을 만들지 않는다.

## 이슈 중요도와 출처 다양성

기사 건수가 많은 이슈를 자동으로 중요한 이슈로 간주하지 않는다. 독립 publisher 수, 신뢰도 가중 확산도, source concentration, 실제 시장 반응, 재전송 제거 결과를 우선한다.

한국장은 국내 수급·환율·업종 자료를 중심으로 하되 해외 핵심 매체가 한국 이슈를 독립적으로 보도했다면 국제적 중요도 신호로 반영한다.

헤드라인만 있는 유료 매체 기사는 authority가 높아도 단독으로 핵심 이슈를 확정하지 않는다. 충실한 공개 요약, 다른 독립 매체, 실제 가격 반응 중 하나 이상이 필요하다.

## 문체 원칙

전문적인 투자 브리핑 문체로 쓰되, 초보 투자자도 흐름을 이해할 수 있게 설명한다. 전문 용어는 문맥 안에서 의미가 드러나게 쓴다.

“확인할 필요가 있다”, “지켜볼 필요가 있다”, “변동성이 확대될 수 있다” 같은 표현은 남발하지 않는다. 쓸 때는 무엇을 어떤 조건에서 확인해야 하는지 구체적으로 붙인다.

특정 종목 매수·매도 조언처럼 쓰지 않는다.
