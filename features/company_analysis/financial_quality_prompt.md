# 재무 품질 분석 프롬프트

기업 분석 보고서의 `Financial Summary` 아래에 `재무 품질 분석` 하위 섹션을 작성한다. 이 섹션은 별도 점수 체계의 이름을 노출하지 않고, 기업의 재무제표가 보여주는 **자본효율성, 현금전환, 성장의 질, 재무 안정성, 자본배분**을 해석하는 용도다.

## 작성 위치

`## 2. Financial Summary` 안에서 핵심 재무 데이터 표와 기본 재무 해석 뒤에 다음 하위 섹션을 추가한다.

```markdown
### 재무 품질 분석
```

## 분석 원칙

- 정량 데이터 표와 구분되게, 숫자를 단순 전사하지 말고 재무의 질을 해석한다.
- 계산 가능한 항목만 사용한다. 다만 완전한 ROIC, WACC, EBITDA가 없어도 `앱 계산 재무 품질 분석`에 영업이익/자산, CFO/영업이익, FCF/매출, 장기부채/CFO, 유동비율 같은 대용 지표가 있으면 이를 사용해 예비 판단을 내린다.
- 이미 계산된 대용 지표가 있는 항목은 `추가 확인 필요`로 되돌리지 않는다. 대신 "현재 확인 가능한 대용 지표 기준"이라고 전제하고, 정확도를 높이기 위해 추가로 확인할 자료를 마지막 열에 적는다.
- 숫자가 전혀 없거나 분모/분자가 모두 확인되지 않는 항목만 `확인 필요`로 표시한다.
- 이 섹션은 매수/매도 추천 점수가 아니다. 좋은 기업인지 판단하기 위한 재무 체력 점검이다.
- 밸류에이션은 별도 섹션에서 다루고, 여기서는 가격이 아니라 기업의 질을 평가한다.
- 금융회사, 초기 성장 기업, 경기민감 산업재는 업종 특성을 반영해 조심스럽게 해석한다.

## 확인 항목

| 항목 | 핵심 질문 | 우선 지표 | 보조 지표 |
| --- | --- | --- | --- |
| 자본수익성 | 투입한 자본 대비 영업이익을 잘 내는가? | ROIC, ROIC 추세 | 영업이익/자산, 영업마진 |
| 현금전환 | 회계상 이익이 실제 현금으로 전환되는가? | CFO/EBITDA | CFO/영업이익, CFO/순이익 |
| 자유현금흐름 | 투자 후에도 현금이 남는가? | FCF, FCF/Revenue | CapEx/CFO |
| 성장의 질 | 성장하면서 수익성과 현금흐름이 함께 좋아지는가? | 매출, 영업이익, CFO, FCF 추세 | 마진 추세 |
| 재무 안정성 | 부채와 유동성 위험이 관리 가능한가? | Net Debt/EBITDA, 이자보상배율 | 현금, 총부채, 장기부채 |
| 자본배분 | 벌어들인 현금을 합리적으로 쓰는가? | CapEx, 배당, 자사주, M&A | FCF 대비 주주환원 |

## 출력 형식

가능하면 아래 표와 해석 문단을 함께 작성한다.

```markdown
### 재무 품질 분석

| 항목 | 예비 판단 | 근거 | 추가 확인 포인트 |
| --- | --- | --- | --- |
| 자본수익성 | 양호/보통/취약/확인 필요 | ... | ... |
| 현금전환 | 양호/보통/취약/확인 필요 | ... | ... |
| 자유현금흐름 | 양호/보통/취약/확인 필요 | ... | ... |
| 성장의 질 | 양호/보통/취약/확인 필요 | ... | ... |
| 재무 안정성 | 양호/보통/취약/확인 필요 | ... | ... |
| 자본배분 | 양호/보통/취약/확인 필요 | ... | ... |

[2~4문장 해석]
```

## 문체

- 한국어로 자연스럽게 쓴다.
- `점수`나 별도 방법론 이름은 쓰지 않는다.
- 확정 판단보다 “현재 확인 가능한 자료 기준”이라는 전제를 둔다.
- 단년도 숫자보다 3년 추세와 현금흐름의 방향성을 우선한다.
